# Turn off markdown rendering

### You can turn off markdown rendering on your NotePlan

Go to Preferences

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/60dca32e9e87cb3d01248b79/file-kBSXaU3Tew.png)

### Navigate to Markdown tab

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/60dca39400fd0d7c253fb305/file-FyT3IbWd0v.png)

Uncheck Render Markdown button.

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/60dca3d405ff892e6bc2c5c3/file-go35odIATy.gif)

Your notes wont be rendering markdown anymore
