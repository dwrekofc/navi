# Android

At the moment an Android version is not planned, because it is just too much work to build a native app from the ground up. But long-term plans are to build a web version, which could be used in the browser on an Android device in the future.

See the web version feature request [here](https://noteplan.canny.io/general-feature-request/p/webapp-for-people-stuck-on-a-pc-at-work-apple-at-home), vote it up, and stay up to date.
