# TestFlight version is not detecting my subscription

If you have installed the TestFlight version of NotePlan (on Mac or iOS) for the first time on a new device (where you never had NotePlan installed before), it might not detect your subscription and lock up editing notes. To solve this first install the AppStore version, then run it so it detects your subscription, and then you can install the TestFlight version.
