# Link Alias

If you want to add a wiki-link to a sentence the problem is often that it doesn't really fit into the flow:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/63ceb1bf26cb8547dcb0d673/file-zp4VZr4EeD.png)A link alias allows you to change the text of the link and make it flow with the sentence:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/63ceb2c3627fa96d949401cc/file-Dn0iiFvECh.png)You can do this by embedding the wiki-link, for example `[[Many tasks are processes]]`, into a markdown link `[title](link)`:

```
[alias]([[wiki-link]])
```

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/63ceb347627fa96d949401cf/file-9UFDJvXarD.gif)
