# Share Files Between Devices

**Update: Since v3.2 NotePlan syncs also file attachments inside notes between devices.**

NotePlan currently only synchronizes images you add to your notes between your devices, but files, such as PDFs are kept locally and are not synced. You can still sync documents and other files using third-party apps and link them into NotePlan.

## KeepIt

[AppStore Link](https://apps.apple.com/de/app/keep-it/id1272768911?l=en&mt=12)

1. Add a file on iOS or Mac to KeepIt so it syncs it across your devices.
2. Open the file in KeepIt.
3. Tap on the share icon at the bottom.
4. Select "Share Item Link".
5. Select NotePlan, choose the day (modify the link if needed), and hit "Save" at the top right.
