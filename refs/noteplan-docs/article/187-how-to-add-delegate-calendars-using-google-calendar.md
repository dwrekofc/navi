# How to add delegate calendars using Google Calendar?

NotePlan syncs with your calendars using Apple's Calendar API (iCal). You can add [accounts](28-how-to-add-new-calendars.md) in the "internet accounts" or calendar accounts preferences of the Apple Calendar app or in the system preferences.

If you don't see your delegated calendars from Google, you can add them in the Accounts settings of the Apple Calendar app:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/63ececf5e669fb108d6ff177/file-KQvhMTWi1q.png)

You may need to restart NotePlan after enabling some calendars.
