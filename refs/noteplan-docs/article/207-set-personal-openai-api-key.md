# How to create and fund your personal OpenAI API Key?

NotePlan uses OpenAI for some features such as transcribing voice input and to manipulate existing text with a custom prompt or transcribing images like handwritten. Each call to OpenAI's services has a small cost. Therefore we rate limit free AI calls (included in the subscription) to a monthly quota.

You can create your OpenAI account and use your API key instead to avoid running into rate limits.

## Create an OpenAI API Key

1. Go to [https://openai.com/](https://openai.com/) and sign up (see top right) to create an account. To see the login button, you need to hover your mouse cursor above the navigation:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6658a3f6dd0f8c60bb3fc75d/file-sCrZFcKeYZ.png)2. Before you can use an API key, you need to fund your account. OpenAI operates with a prepaid system now. Click on the settings button top right in the navigation.

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6658a45f5940934e140f418d/file-B3apEBZeBT.png)3. Click in the left sidebar on "Billing" and find the "Add to credit balance" button in the center of the screen, after adding your payment details.

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6658a4d4dd0f8c60bb3fc75e/file-NkbsARFfXD.png)

4. Once you have funded your account, go back to the dashboard (see top button right). Then click on "API Keys" in the left sidebar.

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6658a55be1989867dcefd541/file-Bb08leDC27.png)5. Find the "Create new secret key" top right.

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6658a5a05173914f806c1236/file-VVo4770hE3.png)

6. Type in the name, set permissions to "All" and create the key clicking on "Create secret key":

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6658a5f15173914f806c1237/file-ZAM7UabcXi.png)7. Make sure to copy the key immediately and store it somewhere safe in case you lose it (you can always delete and re-create new.

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6658a6455173914f806c1238/file-lecP25Ng3Q.png)

8. Copy & paste the key into NotePlan.

On iOS you'll find "AI Responses" in the preferences under "General", tap on "Manage". On macOS, follow [this guide](213-where-do-i-enter-my-openai-key.md).
