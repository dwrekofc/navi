# How to empty the Trash folder

On Mac, you can right-click on the “Trash” item in the left sidebar, then click on "Empty Trash".

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/61e308161adf855680c7a1c0/file-vBWgUPUWSs.gif)

On iOS, you need to swipe “Trash” in the sidebar, then you can tap on “Empty Trash”:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/60ea8efc61c60c534bd6e4c5/file-EBBKuxrBPU.gif)
