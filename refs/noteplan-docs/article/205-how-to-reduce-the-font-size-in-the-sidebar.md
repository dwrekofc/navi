# How to reduce the font-size in the sidebar?

On Mac, the sidebar in NotePlan follows the size of your system setting "Sidebar icon size". Open the system settings and search for "Sidebar", then select "Sidebar icons" and change it. It should update the font-size in NotePlan live:
