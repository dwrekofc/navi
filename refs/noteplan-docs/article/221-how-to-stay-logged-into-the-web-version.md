# How to stay logged into the web version?

When you log in using the web version, you will notice that you get logged out automatically the next day. The solution is to check "Keep me signed in" while you are logging in, but this box won't appear if yo are using Touch ID for your login. When you see the Touch ID login, click on "Cancel" instead:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/65de46e30f2a4c04f24d9a71/file-b5Vnc95G82.png)Now you will see the checkbox "Keep me signed in":

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/65de471ce1b8450992791e05/file-NW11L92Izf.png)Check it and continue your usual sign-in with email and password.
