# Hidden Settings

NotePlan has a list of hidden settings that can be enabled/disabled through the Command Bar. The reason they are hidden is to simplify the settings menu and not overload it with settings that will be used rarely by the majority of users.

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/65a143727273d90e4339b462/file-biR33RZhM6.png)

To access a hidden setting search for the keyword (see below) in the Command Bar and click/tap on it in the results.

## Settings

1. Set OpenAI API Key
2. Enable/Disable OpenAI
3. Enable/Disable Hidden Wiki-Link Brackets
