# How to fix: Tasks appear when I write bullets

By default, if you are typing a dash (-) at the beginning of a line NotePlan creates bullet points. If you are typing an asterisk (*), tasks are created and checklists will appear if you type a plus (+).

If this is not what you see or if you want to change this behavior, check the **Preferences > Markdown > Recognize as...:**

**![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6455092451152c21f5041689/file-gJQuQIAqA5.png)**

In the screenshot above you see the default settings. If you have selected "Recognize - as Todo", it will create a task when you type a dash at the beginning of a new line. Remove the check if you don't want this to happen.
