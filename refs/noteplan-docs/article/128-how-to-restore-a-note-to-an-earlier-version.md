# How to restore a note to an earlier version

NotePlan saves the change history of your notes and stores them locally in a database (not synced across devices). This means each device may contain a different revision history. The device where you have edited a note would contain the most detailed revisions. The database will be trimmed and old revisions will be deleted over time, so it doesn't grow too large.

You can access the revisions of a note by selecting "View Revisions" from the note menu:

**macOS**
Open the note you want to restore and click on the menu button top right, then on "View Revisions".
