# Archive processed tasks

This moves all to-dos, which are marked as done, under a new `## Done` title (if not already there).

This cleans up your list and makes it easier to see what’s still open. It also feels really good to sweep out the checked-off tasks and see the list shrinking.

To archive and open the tasks, click 3 dots icon:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/611bf1d3b55c2b04bf6df2fd/file-vqDpjafitZ.png)

Then select Archive Processed Tasks

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/611bf05ab37d837a3d0e4976/file-uf7mnNH2ws.png)
