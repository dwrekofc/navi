# Install Plugins on iOS

On iOS you can install plugins using the Command Bar. Open it by tapping on the magnifying glass bottom right of a note or hit CMD+J on iPad.

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/61ec546f8200bc052eb8186c/file-559vfWF1Of.png)

Then type `/install plugin` and select the suggestion and select the plugin you want to install after it got loaded:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/61ec556439e5d05141b620e4/file-yNrfQ18Faq.gif)
