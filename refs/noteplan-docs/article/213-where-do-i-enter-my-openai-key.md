# Where do I enter my OpenAI Key?

You can open the command bar on iOS (tap on the search icon in the toolbar or bottom right) or on Mac (CMD+J) and type "Set AI API Key". Then select the "Set AI API Key" option:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/68714a22be3db56c42058414/file-Exsv9ywCjT.png)

If you don't have a key, [read this](207-set-personal-openai-api-key.md). Note: A chatGPT subscription doesn't fund your OpenAI account. These are separate payments.
