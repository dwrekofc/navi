# Change keyboard shortcuts

You can in general change the shortcuts of all Mac apps available through the menubar. Read Apple's article for a step-by-step tutorial: [Create keyboard shortcuts for apps on Mac](https://support.apple.com/guide/mac-help/create-keyboard-shortcuts-for-apps-mchlp2271/mac).

In short, go to System Preferences > Keyboard > Select Shortcuts > App Shortcuts > Add NotePlan > Write the title of the menu item > Define your shortcut:
