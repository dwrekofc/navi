# Where is the toolbar on iPad?

If you are using an iPad with an external keyboard, and you are searching for the toolbar to format text, add headings, and other elements like tables, you might not recognize it at first. This is because the toolbar buttons are merged with the shortcut toolbar (where you can auto-correct or fill words in your text). It shows up at the bottom and only when you are actually editing text:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6407445d9c8683055bad3832/file-ZFhG399CyZ.png)

If you don't see this toolbar, make sure you have enabled Shortcuts and possibly other options in the iOS preferences:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/64b5482378d5c33f49241045/file-B5qcGsRtbS.jpg)
