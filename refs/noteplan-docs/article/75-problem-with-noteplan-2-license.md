# Problem with NotePlan 2 license

You have purchased NotePlan 2 outside the AppStore and received a license code via email. Now your license stopped working when you enter it into NotePlan 2?

Most probably you have activated the license too often without deactivating it on your old Mac. To fix this problem, write us at [[email protected]](cdn-cgi/l/email-protection.md) (or click the bottom right on the message widget).

Send us your license code (as text, so we can copy it quickly) and ideally mention in your subject line something like "NotePlan 2 license problem", so we can quickly see and prioritize your request.
