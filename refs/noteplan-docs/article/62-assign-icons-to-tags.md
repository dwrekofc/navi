# Assign icons to tags

NotePlan doesn't support adding icons directly to tags, but you can simply use an emoji in the name of a tag:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6089b28b8af76a714bfdc1ff/file-DHAsWM6nAq.png)

While writing the tag, hit Control+CMD+Space to open the emoji picker and choose one. It will be inserted into the text. Next time you need the tag, just type the text, in this case, "followup" and select it from the autocomplete menu. The tags will be also visible in the sidebar.

You can also consider using nested tags and use the emoji as the root item by adding a slash after the emoji:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6089b32af8c0ef2d98df75cd/file-7tKEeVJBxW.png)

The sidebar will display a drop-down menu with the nested structure:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6089b34cf8c0ef2d98df75ce/file-eUshyMSG6n.png)

You can also add emojis to @tags and notes, and folders.
