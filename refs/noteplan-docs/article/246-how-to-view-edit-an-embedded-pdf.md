# How to view & edit an embedded PDF

To view a PDF you can double click on the embedded PDF and it opens in preview. However, the Preview app doesn't save the PDF is you edit. To workaround this, you can use a different app to edit the PDF such as Nitro PDF Pro. It can edit and save the PDF.

[Read this](172-add-file-attachments-and-images-to-notes.md) to learn how to add a PDF to your notes.

## How to change the default PDF app?

1. Find any PDF in Finder, right-click and select "Get Info":

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/67813eee3364dc5ef1512041/file-LJyMyWrO6d.png)2. Change the "Open With" selection to some other app like "Nitro PDF Pro":

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/67813f393364dc5ef1512045/file-DCg3jrxouV.png)3. Click on "Change All" and accept:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/67813f81df00b07b83d5be52/file-t7v5c17nxX.png)4. Now you should be able to edit the PDF by double clicking it in NotePlan, then open it in the custom app:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/678140173364dc5ef151204a/file-TvDl90I8qP.png)
