# How to plan with the Kanban Board

The [Cards view](239-card-kanban-view.md) lets you build a Kanban board, and this allows you to use your notes to plan your projects in advance. In this article we go through a few features that helps you use this feature.

## 1. Open Notes in Split View

The easiest way to see what's planned (the Kanban board) while editing and making changes is to open a note in split view. You can do this throughout NotePlan by pressing ⌥ (option) and clicking on the note. This will open that specific note on the right side of the Kanban board. Adjust the width so you can still see the cards.

Alternatively, you can open notes with ⌘+Click in a new window.

## 2. Add and enable/disable Fields

To make cards more meaningful, you can add more visible details using fields. Using [Frontmatter](237-frontmatter.md), add fields like "department: Marketing", or "assigned: Eduard".

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/679421a98b694b5deb8857d1/file-gIUMvI2ji4.png)

Use the "Fields" button in the toolbar to enable and disable fields.

## 3. Update values directly

You can click on the fields to change their values in each card. Make sure you "seeded" enough values in other cards, so you got options.![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/679422f603fa1a07951102a1/file-TjWs9wpeSk.gif)
