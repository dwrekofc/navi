# Use >today for recurring tasks

If you create a task and attach the >today tag, it will always stay at the top of today's note. Instead of checking off this task, drag it and drop it while keeping the option key pressed. This will create a copy that you can check off. The best approach is if you create a dedicated note where you store recurring tasks:
