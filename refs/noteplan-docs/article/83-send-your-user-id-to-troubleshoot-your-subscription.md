# Send your User ID to troubleshoot your subscription

If you are facing issues with your subscription, you can send us your User ID, so we can look up your current status in our system which tracks purchases anonymously (only with this User ID we can associate it with an email).

Below is how you can look it up and copy it.

**On Mac**

In the preferences, go to "Licence" and click on "copy":

**![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6202b8f468cd260cc2d3902c/file-IsEUZfdIFQ.png)
**

**On iOS**

Open the Settings in the left sidebar (scroll down), then tap on "Subscription Details" and click on "Copy User ID":

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/678e549abff6cd496153c192/file-fDOlemidjV.gif)Now you can copy the User ID:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6202b9a6d86136157d9a2163/file-RcdZthOHlf.png)
