# Web Version: Trial Banner appears despite subscribed Apple ID login

If you login successfully into the web version (https://app.noteplan.co) using the Apple ID option (with subscription), but still see a trial banner at the top, review your VPN, or browser settings (an extension like an ad blocker might also create issues) if you are using any to whitelist the following sites or the full domain:

- "https://stripe.com"
- "https://r.stripe.com"
- or domain "stripe.com",
- and "https://api.noteplan.co", "https://app.noteplan.co"
