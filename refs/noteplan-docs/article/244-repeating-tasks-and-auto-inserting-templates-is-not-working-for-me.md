# Repeating Tasks and Auto-Inserting Templates is not working for me

If you are using auto-insert templates and repeat a task, NotePlan will insert the template and then insert the repeated task for you. If you have trouble to get this to work, most likely the calendar note already exists as file and NotePlan will not attempt to insert the template again.
