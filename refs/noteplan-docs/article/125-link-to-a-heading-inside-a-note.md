# Link to a heading inside a note

You can not only link one note to another using `[[note name]]`, but NotePlan also lets you link directly to any heading in another note.

1. Start typing `[[`,
2. then type the partial name of the note title,
3. use the arrow keys (**not** the mouse) to highlight which note you want to link to, but don't hit enter yet,
4. and then type `#` to see all available headings
5. select a heading and hit enter or use the mouse to select one.

For example, linking to `[[Acme Project#Timeline]]` lets you jump directly to the "Timeline" heading inside the "ACME Project" note. Heading links are great to move faster and link to specific sections in longer notes without having to scroll through the entire document.
