# Search text inside a Note

If you want to find some specific text inside your note (general or calendar note) first open the note and then hit `⌘+F` on Mac. A search form will open up at the top and you can type any terms you want to search for. While you type it will start to search for this text.

You can press `ENTER` to switch through the results and `ESC` to exit the search alternatively to the buttons.
