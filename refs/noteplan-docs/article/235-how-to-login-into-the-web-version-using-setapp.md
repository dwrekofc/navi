# How to login into the web version using Setapp?

If you are using the Setapp version, it syncs like the App Store version with iCloud (CloudKit). Your iCloud data is linked to your Apple ID. To access your data on the web, using the [web version](https://app.noteplan.co), you therefore need to login with your Apple ID, picking the Apple login. Your subscription, nor your data will be detected if you login with the Email option.
