# How to fix TestFlight: "Developer has removed the build"?

If you are a long-time TestFlight user of NotePlan, you might see a message "The developer has removed the build". To fix this, you can scroll all the way down and click on "Stop Testing" and join the TestFlight group again by clicking on this link: [https://testflight.apple.com/join/fm9q4OjE](https://testflight.apple.com/join/fm9q4OjE)
