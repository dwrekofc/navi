# How to Create a Teamspaces Database Log File

For troubleshooting purposes it can help to see how your notes look like in the database. This helps to spot issues. The log file is a spreadsheet that lists all your notes (without content) and some other metadata.

You can create a database log file in the Settings > Lab, below the Teamspace login:
