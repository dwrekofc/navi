# Sign in with Apple

You can sign in with Apple in the license detail screen on macOS or iOS so all your devices have the same ID and can be linked to a Stripe subscription:

**macOS**
Open the settings (CMD+, or see "NotePlan" > "Settings..." in the menubar), then click on "License".
*Note: Signing in is not available in the TestFlight Beta. Use the App Store version to sign in, then return to the TestFlight version.*

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6411dd7b2291250577b83b16/file-Kqw7Wnv5pE.png)

**iOS
**In the left sidebar scroll to the bottom and tap on "Settings", then on "Subscription Details" at the top. On this screen you should see a button "Sign in" / "Sign out".

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/678e549abff6cd496153c192/file-fDOlemidjV.gif)

Once you sign in, the User ID at the bottom changes. Make sure to sign in on every device you want to use NotePlan on. To connect and sync between devices, you need to use the same Apple ID.
