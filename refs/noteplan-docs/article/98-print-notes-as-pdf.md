# Print notes as PDF

You can print all of your notes in PDF with macOS, iOS or iPadOS built-in print feature. Just follow the steps below to create a PDF export of your notes.

#### **MacOS:**

#### **Open the note that you want to print and go to File > Print...**

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/60ddfcc300fd0d7c253fbb93/file-f9Seq2cZll.png)

#### After the print box opens, click the PDF button located in the bottom left corner of the Print dialog box and select "Save as PDF"

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/60ddfd219e87cb3d01249441/file-liUi94SJIY.png)

#### **iOS and iPadOS:**

#### **Open the note that you want to print and select "Export as Rich Text (RTF)"**

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/642fc9d67bf4bb61c01170b5/file-cNgH64Dd0Q.jpg)

#### Tab the Share button and select "PRINT":

#### **![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/642fca1b1274e915f2f92fbe/file-ouZlKUEyjv.png)**

#### Tab the Share button again on the print window and select "Save to Files"
- a PDF copy will be saved to your files:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/642fca9191efbc3c6264a6c6/file-VM5SdSlCax.png)
