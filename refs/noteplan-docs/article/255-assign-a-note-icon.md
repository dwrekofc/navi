# Assign a Note Icon

*Note: Available from v3.16.3*

You can assign icons to notes in the [Frontmatter Editor](https://noteplan.co/changelog/v3.16.3-frontmatter-editor) which are visible in the left sidebar and in the notes overview when opening a folder. In addition to defining an icon, you can set the style (regular, solid or light) and the icon color.

Following options are available:

- `icon`: Choose an icon from the list. You can type a name to search like "book"
- `icon-color`: Choose a color for your icon, you can use hex codes or select a color from the list along with the shade like "sky-400"
- `icon-style`: Choose between regular, solid or light. This changes the appearance of the icon
