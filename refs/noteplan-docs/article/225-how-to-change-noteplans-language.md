# How to Change NotePlan's Language

NotePlan is currently available in German and English versions. If you want to switch between them and do not have either as your system language, you can do so via your Mac's settings.

If you go into your Mac's System Settings under "Language and Region" there is a setting where you can change the language for an individual application.

![](../../d33v4339jhl8k0.cloudfront.net/inline/124420/a013702d824ad36d01c9e97be945c655ba433482/3a59c8cba23b1d0c64934738d202043ffb601a70/CleanShot-2024-01-05-at-16-52-33.png)

From there, you should be able to change NotePlan's language to English from German or vice versa.
