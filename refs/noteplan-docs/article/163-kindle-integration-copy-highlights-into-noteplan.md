# Kindle Integration (Copy Highlights into NotePlan)

The following integration works with a Chrome Extension that adds buttons to your Kindle Highlights at [https://read.amazon.com/](https://read.amazon.com/) to copy individual highlights with all of the metadata into a new note in NotePlan.

[→ Install the Chrome Extension and then the user script here.](https://github.com/jlc467/np3_user_scripts)
