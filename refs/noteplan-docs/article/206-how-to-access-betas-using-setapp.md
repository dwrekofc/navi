# How to access betas using Setapp?

**Note: Beta versions are exclusive for App Store subscribers at the moment.**

You need to enable beta versions in Setapp's settings before you can download NotePlan's betas. Before you download the beta, it's always recommended to create a [backup](32-how-to-make-a-backup-of-my-notes.md).

1. Open Setapp and click on the settings icon:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/649d942dcfd7fe604a7fe3d9/file-KlausLSnwv.png)

2. Under "Updates" enable Betas:

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/649d9483cfd7fe604a7fe3dc/file-TxKBxLPDBq.png)

Now you can search for NotePlan and install the beta.
