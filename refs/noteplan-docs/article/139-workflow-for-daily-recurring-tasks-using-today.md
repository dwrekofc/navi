# Workflow for Daily Recurring Tasks using ">today"

Do you have daily recurring tasks? Such as creating content, reviewing sales metrics or you want to get a daily habit started.

Try the following daily workflow:

1. Schedule a task from a regular note into today using the `>today` tag.
2. Drag & drop the task from the references into your daily note holding the Option key.
3. It creates a copy of your task which you can complete.

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/6283c3308bf21a2e2b6d4e68/file-i4fC7hH63a.gif)

By seeing the recurring task in your reference every day you get a gentle reminder to complete it. Instead of checking the task off in its original note, you create a copy of it by dragging the task and holding down the Option key while dropping it into your note. Then you can complete the copy instead of the original and have a record of your work.
