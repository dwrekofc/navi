# Open Notes in Split View / New Window

## **Split View**

On macOS you can open any note in a split view by pressing control (⌃) and clicking on the note (including notes in the sidebar, in the command bar, calendar, etc.). Alternatively right click the note in the left sidebar and select "Open In Split View".

On iOS only the iPad version allows you to open a note in a new window/split view. You can long-press a note in the sidebar to reveal the option.

## New Window

On macOS you can open any note in a new window by pressing command (⌘) and clicking on the note (including notes in the sidebar, in the command bar, calendar, etc.). Alternatively right click the note in the left sidebar and select "Open In New Window".

The note menu top right also allows you to open a note that you have already opened in a new window:
