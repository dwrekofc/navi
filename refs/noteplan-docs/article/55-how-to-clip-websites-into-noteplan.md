# How to clip websites into NotePlan?

NotePlan has no web clipper, but you can use the tool [MarkDownload](https://apps.apple.com/gb/app/markdownload/id1554029832?mt=12) to extract a markdown version from a website and import it into NotePlan.

Read here [How to import notes](33-how-to-import-notes.md).
