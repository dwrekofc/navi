# Insert templates into existing notes

You can fill empty notes with templates using the "Insert Template" button, but how do you insert a template into an existing note?

- You can type "/insert template" anywhere inside an existing note.
- Using the command bar (CMD+J), type "insert template" and then select the template to insert it where your cursor is located.
- Tap on the + button (on Mac) when you hover over an empty line and select "insert template".
