# How to send a crash report?

You can send us a crash report if NotePlan is crashing for you. A crash report helps us to find the source of the crash and make fixes relatively fast.

## [On Mac](#mac)

1. Open the app "Console" (search with Spotlight).
2. Click on "Crash Reports" in the left sidebar.
3. Find NotePlan in the list.
4. Send us the exported crash report file to [[email protected]](cdn-cgi/l/email-protection.md).

![](../../cdn.filestackcontent.com/Screen%20Shot%202021-04-21%20at%2021.23.08.png)

## [On iOS](#ios)

1. Open Settings app.
2. Go to "Privacy", then "Analytics & Improvements".
3. Select "Analytics Data".
4. Locate the log for NotePlan. The logs will be named in the format: <AppName>_<DateTime>_<DeviceName>
5. Select the desired log and tap on the share button top right.
6. Send use the log to [[email protected]](cdn-cgi/l/email-protection.md).
