# How to create an event or reminder?

If you have trouble creating an event or reminder check first if you have [added your calendar to macOS/iOS](28-how-to-add-new-calendars.md) and if NotePlan has [access to your calendars](77-events-and-or-reminders-are-not-displayed.md).

## On Mac

Click on any time slot on the right sidebar displaying a timeline. A dialog should open where you can define the details of your event. Fill out at least the title, check if the time is right, and hit "Save":

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/628d1694c01fce37d9b162b0/file-TkrLYAMHJo.gif)

## On iOS

Tap on the menu icon top right inside a daily note, then "New Event / Reminder":

![](../../d33v4339jhl8k0.cloudfront.net/docs/assets/6081f7f4c9133261f23f4b41/images/628d16e6c01fce37d9b162b3/file-Zit2se4vvp.png)
