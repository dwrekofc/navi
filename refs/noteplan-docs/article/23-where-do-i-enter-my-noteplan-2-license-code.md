# Where do I enter my NotePlan 2 license code?

If you have purchased NotePlan 2 from the NotePlan website (not the AppStore) and received a license code email, first make sure you have downloaded NotePlan 2 from the website: [Download here](https://noteplan.co/download).

After starting NotePlan 2 you can enter the license code either in the purchase screen or click on "NotePlan 2" in the menubar, then "Enter License".

If you have purchased NotePlan 2 from the AppStore, you don't need a license code. Just re-download it from the AppStore.

NotePlan 3 doesn't need a license.
