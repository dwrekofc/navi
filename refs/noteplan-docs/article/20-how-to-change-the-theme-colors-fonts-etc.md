# How to change the theme (colors, fonts, etc.)?

If you want to modify the way NotePlan displays your text and the overall colors used across the app, you can change the theme in the preferences under "Themes":

![](../../cdn.filestackcontent.com/Screen%20Shot%202021-04-21%20at%2021.11.41.png)

In case none of the existing themes are what you are looking for or you want to expand the Markdown support of NotePlan (like strikethrough or underline), you can pick one and modify it. Hover over the theme you want to use as a base, then click on "Copy & Customize":

![](../../cdn.filestackcontent.com/CleanShot%202021-04-21%20at%2021.03.05.jpg)

NotePlan will copy the theme into the "Plugins" folder and open Finder at the saved location, so you can open it with any editor and start customizing. Recommended editors are Sublime Text, Atom, or Visual Studio Code.
