# Embed files into a note

You can embed or link a file into any. Embedding will copy the file into NotePlans folder and be synced across devices (make sure the file is not too big, this would slow down sync).

Linking will keep the video in its original place. You can choose between embedding and linking when dragging a file into the editor.
